题目描述

77

Given two integers n and k, return all possible combinations of k numbers out of 1 … n.

For example,
If n = 4 and k = 2, a solution is:

[
[2,4],
[3,4],
[2,3],
[1,2],
[1,3],
[1,4],
]

解题思路

这题是求组合数，对于permutation和combination这类的题目，都可以用回溯递归或者动态规划来做。
对于每个回溯，若要从n个数字中挑选出k个数，可以：
从start到n挑选k个数，
从start+1到n挑选k个数，
…
从n+1-k到n挑选k个数（此时只有一种挑选方案）。

也可以用动态规划的思想:
$$c(n,k) = c(n-1,k) + c(n-1,k-1)$$
也就是取n的情况（还剩下k-1个数要取），不取n的情况，此时还剩下k个要取。
