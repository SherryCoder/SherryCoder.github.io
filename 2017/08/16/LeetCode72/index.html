题目描述:

Given two words word1 and word2, find the minimum number of steps required to convert word1 to word2. (each operation is counted as 1 step.)
You have the following 3 operations permitted on a word:
a) Insert a character
b) Delete a character
c) Replace a character

解题思路:

使用Dynamic Programming。
这题是将一个字符串以最少的步骤变换到另外一个字符串。可能是太久没有做动规类题了，我想了一会儿并没有想到用矩阵的方式。一个字符串word1的长度为len1，另一个字符串word2的长度为len2，要将word1转换为word2，可以有三种途径：
1)将len1-1变换到len2然后再+1，
2)将len1变换到len2-1再+1，
3)将len1-1变换到len2-1然后再根据最后一个字母相等否判断要不要加一。
可以依此来向前推计算，因此可以用一个矩阵board来存储，矩阵board的元素(i,j)代表word(0,i)从0开始长度为i的子串转换到word2(0,j)的最小变换距离。board[i][j]可以通过三种方法来获取：
1)board[i][j] = board[i-1][j] + 1，
2)board[i][j] = board[i][j-1] + 1,
3)board[i][j] = board[i-1][j-1] + word1[i]==word2[j]?0:1
边界的初始化需要注意下，如board[i][0]和board[0][j]，这个时候两个串中一个串是空，因此需要执行的最小步骤是另一个字符串的长度。
